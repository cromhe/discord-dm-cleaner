# dm cleaner

wipe ur msgs from dms. use with caution.

> [!WARNING]
> **educational purposes only.** self-botting is against discord tos. use at ur own risk. im not responsible for banned accs.

## setup
1. `pip install -r requirements.txt`
2. put ur token in `config.json`.
   - to get token: open discord web -> f12 -> network tab -> type something -> look for 'authorization' header. ( old )
3. run `python main.py`.
4. enter the channel id (right click dm -> copy id).

## config
- `delay`: seconds to wait between deletes. default 1.5s (recommended).
